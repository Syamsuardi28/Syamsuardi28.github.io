Sistem Absensi Mahasiswa

PHP MySQL Bootstrap

Deskripsi Project

Sistem Absensi Mahasiswa adalah aplikasi berbasis web yang dikembangkan menggunakan PHP native dan MySQL. Aplikasi ini dirancang untuk memudahkan proses pencatatan kehadiran mahasiswa di kelas dengan sistem login multiuser yang membedakan hak akses antara admin, dosen, dan mahasiswa.

Fitur Utama

  • Login multiuser dengan role Admin, Dosen, dan Mahasiswa
  • Dashboard interaktif untuk monitoring kehadiran
  • Pencatatan absensi real-time dengan validasi
  • Laporan kehadiran dalam format Excel dan PDF
  • Manajemen mata kuliah dan jadwal perkuliahan
  • Notifikasi otomatis untuk mahasiswa yang sering absen

Teknologi yang Digunakan

Project ini dibangun menggunakan teknologi PHP native untuk backend logic, MySQL sebagai database management system, Bootstrap untuk tampilan responsive, dan JavaScript untuk interaksi dinamis di sisi client.
